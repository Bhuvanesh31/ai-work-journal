AgentOps Core — July 2026

Built out the full ingestion-to-verification pipeline for capturing AI coding sessions: a PostgreSQL-backed FastAPI service that discovers Claude Code (and sub-agent) transcripts, normalizes them into events, and exposes a read API and minimal UI for inspecting runs. This turns raw local transcript files into a queryable record of session activity — repo attribution, timestamps, token usage, and event drill-down — culminating in a git-commit reconciliation feature and prep for public release. 141 sessions and 67 commits went into hardening this from a bare schema into an end-to-end observability tool for AI-assisted development.

Key achievements:

  • End-to-end capture pipeline: git-remote-based repo resolution, transcript discovery (including nested sub-agent transcripts), retry-hardened ingestion API, and a catch-all + cwd-override path for sessions without a git remote, so no session is silently dropped.
  • Read surface + verification UI: `GET /runs`, `/runs/{id}`, `/runs/{id}/events`, and `/overview` endpoints backed by incremental token-usage tracking and run time-bounds derived from event timestamps, plus a lightweight `/ui` for browsing runs and per-project token charts without a separate frontend.
  • Data integrity tooling: idempotent repository upserts, a reclassification command to re-point catch-all runs once their real repo is known, and `app.maintenance` backfill commands for usage metrics and run time bounds — closing gaps in historical data as the schema evolved.
